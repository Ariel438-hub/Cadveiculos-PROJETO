Atividade de cadastro de veículos desenvlovido em Java na disciplina de Paradigma de Linguagens de Programação.  

Sistema simples em Java para gerenciamento de veículos via terminal, utilizando ArrayList. 
Permite cadastrar, listar, buscar, editar e remover veículos, com validação para evitar nomes duplicados. 
Exibe o total de veículos nas operações, trata lista vazia com mensagens apropriadas e ordena os dados manualmente em ordem alfabética.
